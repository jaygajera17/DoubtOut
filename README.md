


## How To Run ?
**for frontend** <br>
(1) 
```sh
  npm i
```
(2) 
```sh
  npm start
```

**for backend** <br>
(1) 
```sh
 cd backend 
```
(2) 
```sh
  npm i
```
(3) 
```sh
 nodemon ./index.js
```
